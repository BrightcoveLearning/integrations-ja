Drupal 7-Brightcoveコネクタ：設定と使用法

はじめに

Drupal Brightcove ビデオコネクタを使用すると、Drupal 内で Brightcove 動画クラウドの動画とプレーヤーを管理し、Drupal ページに動画を簡単に埋め込むことができます。このページでは、Drupal 7 内で Brightcove モジュールを設定する方法について詳しく説明します。Drupal 8 については、 Drupal-Brightcove コネクタを参照してください。[構成]。

Drupalのインストールは非常に簡単にうまくいく複雑なプロセスです。続行する前に、コンテンツがバックアップされていることを確認してください。

Brightcoveアカウントに接続する

  1. Brightcoveアカウントに接続するには、をクリックしてBrightcoveAPIページに移動します。 設定、次に下にスクロールして ブライトコーブの設定 下のオプション メディア。 これにより、[Brightcove設定] ページに移動します。新しいBrightcoveAPIクライアントを追加するには、をクリックします。 APIクライアント タブをクリックし、 APIクライアントを追加する。 これにより、API クライアントに追加するフォームが表示されます。
  2. ラベルは何でもかまいません。ただし、ブライトコーブのアカウント名と同じ名前を使用すると、識別が容易になります。
  3. アカウント ID を見つけるには、Brightcove Studio  ダッシュボードの左上隅を参照してください。
  4. クライアント ID とクライアントシークレットを取得するには、アプリケーションを登録する必要があります。Brightcove Studioで、次の場所に移動します API 認証 をクリックします 新しいアプリケーションを追加する。
  5. 名前と短い説明を入力します。
  6. アカウントを選択します。下の写真は、すべての必要な権限です。
  7. 保存を押してください。
  8. クライアント ID とクライアントシークレットが表示されます。 クライアントシークレットは、必ず安全な場所にコピーして保存してください。再度アクセスすることはできません。
  9. すべてのIDとキーを入力します。 保存を押してください。
  10. 問題が発生した場合は、日付と時刻が設定されていることを確認し、もう一度試してください。

ブライトコーブの動画フィールドの追加

Drupal 投稿に Brightcove 動画を追加するには、まず新しいコンテンツページに動画フィールドを追加します。これを行うには、[構造] に移動し、[コンテンツタイプ] をクリックします。

Brightcove動画を追加するコンテンツタイプから [フィールドの管理] を選択します。このプロセスは、コンテンツタイプごとに繰り返すことができます。

[新しいフィールドを追加] をクリックします。ラベルを付けて、[フィールドタイプ] に [ブライトコーブ]、ウィジェットの [動画 (アップロード/参照)] を選択します。保存を押してください。

これで、ビデオフィールドが表示されます。

動画の投稿と管理

Drupal 8 とは異なり、Drupal 7 コネクタには個別のコンテンツ管理システムがないため、ほとんどのコンテンツ管理は Brightcove Studio サイトで実行する必要があります。ただし、新しいコンテンツページから、[参照] を押すとすべての画像を表示したり、新しい画像をアップロードしたりできます。

投稿に既存の動画を添付する方法は 2 つあります。動画のタイトルがわかっている場合は、Brightcove フィールドに入力し、オートコンプリートで動画を検索できるようにすると機能します。参照ボタンを押して、すべてのビデオを表示して選択することもできます。

アップロードボタンをクリックすると、アップロードフォームが開き、Drupalから直接新しい動画を追加できます。投稿を公開すると、ビデオが表示されます。
