Drupal 7-Brightcoveコネクタ：インストール

はじめに

Drupal-Brightcove コネクタを使用すると、Drupal 内で Brightcove 動画クラウドの動画とプレーヤーを管理し、動画を Drupal ページに簡単に埋め込むことができます。この記事では、特にDrupal 7プラグインをインストールする方法について説明します。Drupal 8 については、Drupal 8 のインストールガイドを参照してください。

Drupalのインストールは非常に簡単にうまくいく複雑なプロセスです。続行する前に、コンテンツがバックアップされていることを確認してください。

インストール手順

最良の結果を得るには、すでにコンポーザーをインストールしていることを確認してください。その方法については、こちらをご覧ください。

  1. PHP-API-Wrapper をダウンロードしてインストールします。に移動するsites/all/libraries/PHP-API-Wrapper
  2. sites/all/libraries/PHP-API-Wrapperコンソールでに移動します。フォルダ内から、composer install.
  3. ここで、ブライトコーブモジュールをダウンロードしてください。また、依存関係のために他のモジュールをロードする必要があります。
    • カオスツール
    • エンティティ API
    • ライブラリ API
    • 日付
    • ファイルエンティティ
    • メディア
    • [ビュー]
  4. modulesすべてのモジュールをフォルダに移動します。
  5. あなたのdrupalサイトに行ってください。「モジュール」ページにナビゲートします。
  6. 次のモジュールを有効にします:ctools、エンティティ API、ライブラリ、日付 API、日付ポップアップ、メディア、メディアインターネットソース、ファイルエンティティ、およびビュー。これらはすべて、ステップ 3 のおかげで利用可能になるはずです。
  7. 日付と時刻を設定してください！そうしないと、ブライトコーブのモジュールは機能しません。

完了したら、[ 構成] に移動できます。
