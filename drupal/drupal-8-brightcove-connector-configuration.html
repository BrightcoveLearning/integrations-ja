    Drupal 8-Brightcove コネクタ:設定

    はじめに

    Drupal Brightcove ビデオコネクタを使用すると、Drupal 内で Brightcove 動画クラウドの動画とプレーヤーを管理し、Drupal ページに動画を簡単に埋め込むことができます。これはDrupal 8のためであることに注意してください。Drupal 7については、 Drupal 7の指示を参照してください。

    Drupalのインストールは非常に簡単にうまくいく複雑なプロセスです。続行する前に、コンテンツがバックアップされていることを確認してください。

    Brightcove アカウントに接続する

    1. Brightcove のアカウントを接続するには、[ 設定 ] をクリックしてBrightcove API ページに移動し、[ メディア ] の [ Brightcove API アカウント ]  オプションまでスクロールダウンします。 これにより、Brightcove API クライアントページに移動します。新しい Brightcove API クライアントを追加するには、[ Brightcove API クライアントを追加 ] をクリックします。
    2. ラベルは何でもかまいません。ただし、Brightcove のアカウント名と同じ名前を使用すると、識別が容易になります。
    3. クライアント ID とクライアントシークレットを取得するには、アプリケーションを登録する必要があります。Brightcove Studio で、[ API 認証 ] に移動し、[ 新しいアプリケーションの追加 ] をクリックします。
    4. 名前と短い説明を入力します。
    5. アカウントを選択します。下の写真は、すべての必要な権限です。
    6. 保存を押してください。
    7. クライアント ID とクライアントシークレットが表示されます。 クライアントシークレットは、必ず安全な場所にコピーして保存してください。あなたは再びそれにアクセスすることはできません。
    8. アカウント ID を確認するには、Brightcove Studio  ダッシュボードの左上隅を確認します。
    9. すべての ID とキーを入力し、[ 既定のプレーヤー ] を選択します。
    10. [保存]をクリックします。これで、API クライアントがリストに表示されます。

    同期化

    最初に、[ 新しいコンテンツ ] ページに移動して [ Brightcove の動画] をクリックすると、動画を追加するためのオプションが表示されませんが、動画を追加するオプションもあります。これは、まず、データを同期する必要があるためです。最初の同期の後、DrupalのChronはすべてを最新の状態に保ちます。手動同期の場合は、上部の [ レポート ] ボタンをクリックし、[ Brightcove ステータスの概要 ] をクリックします。

    [ Brightcove のステータスの概要 ] ページで、[ すべて同期 ] をクリックします。

    もう一度 Brightcove 動画ページに移動すると、動画が利用可能になります。

    権限の設定

    これで動画を表示および投稿できますが、管理者以外のユーザーが動画を操作する場合は、権限を設定する必要があります。これらの権限を設定するには、まず [ ユーザー  ]、[権限] の順にクリックして、[ 権限 ] ページに移動します。

    下にスクロールして Brightcove の権限を探します。必要に応じて有効または無効にします。Brightcove の権限は自動的に管理者専用に設定されるため、他のユーザーに動画を表示させる場合は、その権限を付与する必要があります。

    これで、コンテンツ管理またはコンテンツの埋め込みに移行できます。
