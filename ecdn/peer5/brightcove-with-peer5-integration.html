プレーヤーに Peer5 プラグインを追加する手順

  1. Studio の [プレーヤー] モジュールに移動して、プレーヤーを選択します (または、既存のプレーヤーを複製して新しい Peer5 が有効なプレーヤーを作成します)。
  2. プレイヤーが作成されたら、プレイヤー名をクリックして編集します。
  3. サイドバーの [ プラグイン]セクションに移動します。
  4. タブ付きメニューで [ スクリプト ] を選択します。
  5. [ スクリプトを追加] をクリックします。
  6. 下のプラグイン文字列をポップアップボックスに貼り付けます (Peer5 アカウントから account_ID を取得します)。 
    https://api.peer5.com/peer5.brightcove.plugin.loader.js?id={Account_ID}
  7. [ 保存] をクリックします。
  8. [ 公開して埋め込み] をクリックし、[ 変更を公開] をクリックします。

統合をテストする

統合をテストする手順

  1. Studioのメディアモジュールに移動します。
  2. ビデオを選択し、[ 公開して埋め込む] をクリックし、[ Web プレーヤー] を選択します。
  3. Peer5 対応プレーヤーを選択します。
  4. [ブラウザでプレビュー] の下のリンクをクリックして、新しいタブでプレーヤーを開きます。
  5. &peer5_show_stats=true URLの末尾に追加します。
  6. Peer5 オーバーレイウィンドウが表示されます。ビデオを再生すると、Peer5 にも送信されているデータが表示されます。
