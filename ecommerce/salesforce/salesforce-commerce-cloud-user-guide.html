    Salesforce コマースクラウドユーザガイド

    このドキュメントでは、Brightcove Salesforce Commerce クラウドカスタムカートリッジを使用して Brightcove Cloud/Server に動画ファイルをアップロードし、必要に応じてストアフロント（製品詳細ページ、カテゴリランディングページ、製品一覧ページなど）にこれらのファイルを公開する手順を説明します。

    複数言語のサポート

    拡張機能は、ビジネスマネージャーインターフェイスの複数の UI 言語サポートを提供します。注：このバージョンの拡張機能では、UI 言語の英語とフランス語がサポートされています。

    動画の追加

    以下は、動画を追加する手順です。

    1. マーチャントツール-> ブライトコーブ-> 動画の追加に移動します
    2. メニューから [ 言語 ] を選択します (デフォルトは、サイトで選択されたデフォルトのロケールです)。
    3. [ビデオ] フィールドの横にある選択オプションをクリックします。
    4. 表示されるダイアログで、ビデオファイルを選択します。
    5. サムネイルフィールドの隣の選択オプションをクリックすると、サムネイルファイルを選択できる同様のポップアップダイアログが表示されます。
    6. ポスターフィールドの横にある選択オプションをクリックすると、同様のポップアップダイアログが表示され、ポスターファイルを選択できます。
    7. フォームを送信して、取り込みプロセスを開始します。
    8. 送信後にフォーム上の成功メッセージを検証します。
    9. 動画が正常に取り込まれたことを確認するには、Brightcove Studio にログインし、Media モジュールに移動します。
    10. また、ビデオをクリックして詳細を表示して、ロケールコードを確認することもできます。

    ビデオの割り当て-カテゴリ

    まず、カテゴリにビデオを割り当てる手順を見ていきます。

    カテゴリへのビデオの割り当て

    1. マーチャントツール-> ブライトコーブ-> 動画の割り当て (カテゴリ) に移動します。
    2. 動画はロケールに割り当てられます。
      • ロケール関連データは、ドロップダウンからロケールを変更することによって表示できます。

        現在サポートされているロケールは、英語 (米国) と francais (フランス) のみです

      • ビデオサムネイルは、Video ID にカーソルを置くと表示できます。
    3. 1 つまたは複数のビデオを割り当てるには、[ビデオの割り当て] オプションをクリックします。
    4. 既に割り当てられているビデオは、割り当てのビデオグリッドで選択したように表示されます。
    5. 割り当てられたスロットは、同じロケールの他の動画には使用できません:
    6. プレーヤーとスロットは、各ビデオタイルで利用可能なドロップダウンを使用して変更できます。
    7. 別のスロットで別のビデオを選択します。
    8. [ビデオを保存] ボタンをクリックして、選択したビデオを割り当てます。
    9. 割り当てられているビデオとスロットを確認します。
    10. カテゴリオブジェクトのデータを確認します。
    11. 出品者ツール → > カタログに移動します。
    12. カタログを選択します。
    13. カテゴリ/サブカテゴリを選択します。
    14. [ 編集 ] を選択します。
    15. 「カテゴリ属性」を選択します。
    16. [ブライトコーブ] セクションまで下にスクロールします。
    17. 動画をサブカテゴリに割り当てるには、カテゴリをクリックします。
    18. サブカテゴリのリストが表示されます。
    19. 手順3～16に従って、動画をサブカテゴリに割り当てます。

    マスター製品へのビデオの割り当て

    1. マーチャントツール-> ブライトコーブ-> ビデオの割り当て (商品) に移動します。
    2. 動画はロケールに割り当てられます。
      • ロケール関連データは、ドロップダウンからロケールを変更することによって表示できます。

        現在サポートされているロケールは、英語 (米国) と francais (フランス) のみです

      • ビデオサムネイルは、Video ID にカーソルを置くと表示できます。
    3. 1 つまたは複数のビデオを割り当てるには、[ビデオの割り当て] オプションをクリックします。
    4. 既に割り当てられているビデオは、割り当てのビデオグリッドで選択したように表示されます。
    5. 割り当てられたスロットは、同じロケールの他の動画には使用できません:
    6. プレーヤーとスロットは、各ビデオタイルで利用可能なドロップダウンを使用して変更できます。
    7. 別のスロットで別のビデオを選択します。
    8. [ビデオを保存] ボタンをクリックして、選択したビデオを割り当てます。
    9. 割り当てられたビデオとスロットを確認します。
    10. カテゴリオブジェクトのデータを確認します。
    11. 出品者ツール-> 商品に移動します。
    12. 製品 ID を検索します。
    13. 製品を選択します。
    14. [ 一般 ] を選択します。
    15. [ブライトコーブ] セクションまで下にスクロールします。

    バリアント商品へのビデオの割り当て

    1. マーチャントツール-> ブライトコーブ-> ビデオの割り当て (商品) に移動します。
    2. プロダクト ID をクリックします。
    3. 選択したマスターのバリアントのリストが表示されます。
    4. 動画はロケールに割り当てられます。
      • ロケール関連データは、ドロップダウンからロケールを変更することによって表示できます。

        現在サポートされているロケールは、英語 (米国) と francais (フランス) のみです

      • ビデオサムネイルは、Video ID にカーソルを置くと表示できます。
    5. 1 つまたは複数のビデオを割り当てるには、[ビデオの割り当て] オプションをクリックします。
    6. 既に割り当てられているビデオは、割り当てのビデオグリッドで選択したように表示されます。
    7. 割り当てられたスロットは、同じロケールの他の動画には使用できません:
    8. プレーヤーとスロットは、各ビデオタイルで利用可能なドロップダウンを使用して変更できます。
    9. 別のスロットで別のビデオを選択します。
    10. [ビデオを保存] ボタンをクリックして、選択したビデオを割り当てます。
    11. 割り当てられたビデオとスロットを確認します。
    12. カテゴリオブジェクトのデータを確認します。
    13. 出品者ツール-> 商品に移動します。
    14. 製品 ID を検索します。
    15. 製品を選択します。
    16. [ 一般 ] を選択します。
    17. [ブライトコーブ] セクションまで下にスクロールします。
    18. サイトのカタログに割り当てられたすべてのバリアントのリストを取得できます。
    19. マーチャントツール-> ブライトコーブ-> ビデオの割り当て (商品) に移動します。
    20. バリアント製品の選択
    21. 5 から 17 までの手順に従います。

    ビデオの埋め込み

    埋め込みビデオステップを使用すると、任意の ISML ページにビデオをレンダリングするためのコードを取得できます。

    1. マーチャントツール-> ブライトコーブ-> 動画の埋め込みに移動します。
    2. 必要な動画とプレーヤーを選択し、[ Get Code ] をクリックします。
    3. ポップアップダイアログがコードとともに表示されます。コードブロックを選択すると、クリップボードにコピーされます。
    4. コンテンツをコンテンツ資産、コンテンツスロット、または任意の ISML テンプレートに貼り付けて、ビデオをレンダリングします。

    一括アップロード

    一括アップロードでは、.csv ファイルをアップロードすることで、動画を一括で取り込むことができます。ジョブは.csv ファイルを処理し、動画を Brightcove アカウントにインジェストします。サンプルの BulkUpload ファイルは、メタディレクトリの下の zip ファイルに添付されています。

    videofile、videoposterおよびは、videothumbnailパブリックにアクセス可能なパスで使用可能である必要があります。
    1. マーチャントツール → > ブライトコーブ → > 一括アップロードの順に移動します。
    2. .csv ファイルを選択します。
    3. [送信] をクリックします。
    4. [ 管理] → [> ジョブ ] に移動します。
    5. 一括アップロードジョブを選択し、「実行」をクリックします。
    6. ジョブ完了ステータスを確認します。
    7. ブライトコーブアカウントで動画オブジェクトを確認します。
    8. 一括アップロードジョブで処理されたファイルは、一時ディレクトリの下の Bulkupload_processedフォルダに移動します。
