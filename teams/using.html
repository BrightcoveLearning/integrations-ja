はじめに

Brightcove Media Managerを使用する前に、チーム用にインストールして構成する必要があります。詳細については、「Teams 用コネクタのインストール」を参照してください。

前のセクションで説明したようにメディアマネージャーをチームに追加した後、ビデオの追加と管理、プレイリストの追加と管理、およびチーム内のメディアマネージャーからのエクスペリエンスの管理を行うことができます。

チーム内のメディアマネージャーから行った変更は、BrightcoveプラットフォームのBrightcoveアカウントに保存され、BrightcoveStudioを介してオンラインで接続するとそれらを確認できます。

ビデオを追加する

  1. チームを開き、以前にMediaManagerをインストールしたチームとチャネルに移動します。
  2. アセットマネージャタブを選択して、メディアマネージャを開きます。
  3. [ 動画を追加 ] タブをクリックします。
  4. アップロードするビデオを指定されたホットスポットにドラッグアンドドロップするか、[ ビデオの参照] をクリックして追加するビデオを選択します。
  5. 追加されると、ビデオはBrightcoveビデオクラウドプラットフォームで処理されます。処理が完了するのを待ちます。
  6. 処理が完了したら、ビデオを編集/管理する準備ができており、以下に示すように「ビデオの編集」リンクをクリックできます。
  7. ビデオの編集ページに移動し、そこでビデオのメタデータを編集できます。

動画を管理する

メディアマネージャーで、[ 動画の管理]タブをクリックします。

ビデオを編集する

  1. ビデオのリストから編集したいビデオを見つけます。
  2. [ 編集]リンクをクリックします。
  3. 必要に応じて、ビデオのメタデータを編集します。
  4. 終了したら、編集ページの一番下までスクロールし、[ビデオクラウドで更新]ボタンをクリックして、変更をBrightcoveビデオクラウドに保存します。

ビデオ埋め込みコードを取得する

  1. ビデオのリストから、埋め込みコードが必要なビデオを見つけます。
  2. [ 埋め込み ] ボタンをクリックします。
  3. ビデオを埋め込むモーダルで、[ クリップボードにコピー]リンクをクリックして、埋め込みコードをコピーします。
    注：必要に応じて、埋め込みのさまざまな構成を設定してその動作を管理したり、コードをコピーする前にプレビューしたりできます。

プレイリストを追加する

  1. メディアマネージャで、[ プレイリストの管理 ] タブをクリックします。
  2. プレイリストのリストの下部にある [ 新しいプレイリストを追加 ] ボタンをクリックします。
  3. 名前と説明を入力し、再生リストの一部として必要な動画を選択します。
  4. [ 作成]ボタンをクリックします。
  5. プレイリストは Brightcove Video クラウドによって処理され、処理後にプレイリストのリストに表示されます。
    注:処理には数分かかる場合があり、プレイリストがプレイリストのリストにすぐに表示されない場合があります。必要に応じて、ブライトコーブを使用して処理を確認します。スタジオ。

プレイリストを管理する

メディアマネージャで、[ プレイリストの管理 ] タブをクリックします。

プレイリストを編集する

  1. 編集するプレイリストを見つけます。
  2. [ 編集]リンクをクリックします。
  3. 編集を行い、[ 保存]ボタンをクリックします。

プレイリスト埋め込みコードを取得する

  1. 埋め込みコードを取得するプレイリストを見つけます。
  2. [ 埋め込み ] ボタンをクリックします。
  3. [ プレイリストの埋め込み]モーダルで、[ クリップボードにコピー]リンクをクリックして、埋め込みコードをコピーします。
    注：必要に応じて、埋め込みのさまざまな構成を設定してその動作を管理したり、コードをコピーする前にプレビューしたりできます。

エクスペリエンスの管理

メディアマネージャーで、[ エクスペリエンスの管理 ] タブをクリックします。

プレビュー体験

  1. プレビューしたいエクスペリエンスを見つけます。
  2. [ プレビュー]リンクをクリックします。

埋め込みコードの経験を積む

  1. 埋め込みコードを取得したいエクスペリエンスを見つけます。
  2. [ 埋め込み ] ボタンをクリックします。
  3. 埋め込みエクスペリエンスモーダルで、[ クリップボードにコピー]リンクをクリックして、埋め込みコードをコピーします。
    注：必要に応じて、埋め込みのさまざまな構成を設定してその動作を管理したり、コードをコピーする前にプレビューしたりできます。

チームのチャンネルタブに動画を追加する

  1. Microsoft Teamsを開き、ビデオを追加するチームに移動します。
  2. 動画を追加するチャンネルを開きます。
  3. [ タブを追加]メニューをクリックします。
  4. 利用可能なアプリのリストからBrightcoveVideoConnectorアプリを選択します。
  5. 構成モーダルで、Brightcoveアカウントを選択します。
  6. 構成モーダルで、埋め込まれたビデオコネクタタブモードとしてクリックします次：
  7. チャンネルタブに追加するビデオを選択し、「保存」をクリックします。
  8. これで、追加されたビデオが[チャンネル]タブに表示されます。

チームのチャンネルタブにエクスペリエンスを追加する

  1. Microsoft Teamsを開き、エクスペリエンスを追加するチームに移動します。
  2. エクスペリエンスを追加するチャネルを開きます。
  3. [ タブを追加]メニューをクリックします。
  4. 利用可能なアプリのリストからBrightcoveVideoConnectorアプリを選択します。
  5. 構成モーダルで、Brightcoveアカウントを選択します。
  6. 構成モーダルで、埋め込みエクスペリエンスコネクタタブモードとしてクリックします次：
  7. チャンネルタブに追加するエクスペリエンスを選択し、[ 保存] をクリックします。
  8. これで、[チャンネル]タブに追加されたエクスペリエンスが表示されます。
