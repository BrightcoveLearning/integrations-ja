はじめに

INXPOは、テレビスタイルのイベント体験を作成するためのビデオストリーミングプラットフォームです。より多くのオーディエンスにリーチするには、ブライトコーブライブを活用してイベントをストリーミングできます。

ステップ

ブライトコーブライブでミーティングを主催するには、次の手順に従います。

  1. Video Cloud Studio で、ライブモジュールを使用してライブイベントを作成します。
    放送完了後にブロードキャストを保存する場合は、[ 完了したらイベントをビデオアセットに変換 ] チェックボックスをオンにしてください。

  2. ライブイベントを作成したら、コントロールパネルからストリーミング RTMP URL とストリーミングキーをコピーします。

    • ストリーミング RTMP URL- ストリーミングエンドポイント URL（エンドポイント URL は Brightcove Live Control Room にあります）
    • ストリーミングキー- ストリーム名（ブライトコーブでは、ストリーム名は常に「生きている」）
  3. Web サイト/イントラネットページにこのライブイベントのプレーヤーを公開して埋め込み、ページの URL を書き留めます。

ストリーミングセッションの設定

ストリーミング RTMP URL とストリーミングキーを使用して、INXPO プロデューサーツールで設定する必要があります。

  1. INXPO プロデューサーツールに移動し、3 つのドットアイコンをクリックして [出力詳細] セクションに移動します。
  2. 鉛筆アイコンをクリックします。
  3. 以下の詳細に、Brightcoveのライブイベントの認証情報を入力します。
  4. [ タイトル ] フィールドに、ブライトコーブのライブストリーム名を追加します。
  5. [ Encoding Profile]フィールドで、エンコーダによるWebキャストの出席者への配信方法を定義するデフォルトのエンコーディングプロファイルを選択します。
  6. [ RTMP URL ] フィールドに、ブライトコーブストリーミングエンドポイント URL を挿入します。
  7. OKボタンをクリックし、それだけです。おまえは生きてる！
