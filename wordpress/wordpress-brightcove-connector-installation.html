WordPress-Brightcoveコネクタ：インストール

このトピックでは、Wordpress 用の Brightcove コネクタのインストールについて説明します。

はじめに

WordPress ブライトコーブビデオコネクタを使用すると、WordPress 内でブライトコーブ動画クラウドの動画とプレーヤーを管理し、WordPress ページに動画を簡単に埋め込むことができます。

インストール手順

  1. WordPress ブライトコーブビデオ接続プラグインをダウンロード
  2. /brightcove-video-connectファイルを解凍して移動する/wordpress/wp-content/plugins
  3. WordPress の [プラグイン] メニューから [Brightcoveのビデオコネクト] をアクティブにする

Brightcove プラグインがサイドバーに表示されている場合は、[ 設定] に移動できます。
